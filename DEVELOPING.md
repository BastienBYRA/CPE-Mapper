# Developing
## Prerequisites
This project tech stack is simple:
| Tech stack | Version |
|---|---|
| Node.js | >= 22 |

That's it!! If you have a Node.js on your system, and an IDE, you're good to go!

## Running
Running the project doesn't require anything specific:

```bash
# Install the dependencies
npm install

# Start the project
node src/cli --help
```

## Testing
To run all tests:
```bash
npm run test
```

For Windows users:
```bash
npm run wtest
```
